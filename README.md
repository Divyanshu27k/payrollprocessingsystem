# 🚀 Enterprise Payroll Processing System (HRMS)

A **production-ready Enterprise Payroll & Human Resource Management System (HRMS)** built using **Java 17, Spring Boot 3 Microservices, React.js, MySQL, Docker, Redis, Kafka, and Spring Cloud**.

The system automates the complete employee lifecycle, including attendance, payroll, leave management, shift scheduling, WFH/WFO tracking, notifications, and reporting. It follows a scalable **Microservices Architecture** with secure REST APIs, JWT authentication, and enterprise-grade best practices.
